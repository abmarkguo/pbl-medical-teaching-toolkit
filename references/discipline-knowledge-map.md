# 跨学科知识图谱

## 概述

本文档提供基础医学各学科的知识图谱和交叉映射，帮助教师在设计 PBL 案例时识别跨学科知识点，构建学科间的知识关联网络，实现多学科整合教学。

---

## 一、基础医学学科知识架构

### 1.1 学科分类与核心内容

| 学科 | 核心内容 | PBL案例典型锚点 |
|------|---------|----------------|
| 解剖学 | 人体形态结构 | 神经走行、血管分布、器官位置、体表标志 |
| 组织胚胎学 | 微观结构与发育 | 组织类型、细胞功能、胚胎发育异常 |
| 生理学 | 功能调节机制 | 心血管调节、呼吸调节、消化吸收、泌尿调节 |
| 生物化学 | 物质代谢与能量 | 糖代谢、脂代谢、蛋白质代谢、核酸代谢 |
| 病理学 | 疾病形态学改变 | 炎症、肿瘤、退行性变、循环障碍 |
| 病理生理学 | 疾病功能机制 | 水电解质紊乱、酸碱失衡、缺氧、休克 |
| 药理学 | 药物作用与机制 | 受体药理、药代动力学、药物相互作用 |
| 医学微生物学 | 病原体与感染 | 细菌/病毒/真菌特性、致病机制、抗感染 |
| 免疫学 | 免疫系统与应答 | 超敏反应、自身免疫、免疫缺陷、免疫调节 |
| 医学遗传学 | 遗传与疾病 | 单基因病、多基因病、染色体异常、基因治疗 |

### 1.2 学科间的知识关联

```
                    医学遗传学
                       ↓
    组织胚胎学 → 解剖学 ← 生理学
         ↓          ↓          ↓
    生物化学 ← 病理学 ← 病理生理学
         ↓          ↓          ↓
    药理学 ← 免疫学 ← 医学微生物学
         ↓
      临床医学
```

## 二、各系统知识图谱

### 2.1 心血管系统

**涉及学科：** 解剖学、生理学、病理学、病理生理学、药理学

**核心知识节点：**

| 知识节点 | 学科 | 关联节点 | 临床连接 |
|---------|------|---------|---------|
| 心脏解剖结构 | 解剖学 | 冠脉走行、瓣膜结构、传导系统 | 心肌梗死、瓣膜病、心律失常 |
| 心肌动作电位 | 生理学 | 离子通道、传导速度 | 心律失常、心电图 |
| 心脏泵血功能 | 生理学 | 心输出量、射血分数 | 心力衰竭 |
| 血管调节机制 | 生理学 | 交感/副交感、RAAS | 高血压、休克 |
| 动脉粥样硬化 | 病理学 | 内皮损伤、脂质沉积、斑块形成 | 冠心病、脑卒中 |
| 心肌缺血 | 病理生理学 | 氧供需失衡、再灌注损伤 | 心绞痛、心肌梗死 |
| 心力衰竭 | 病理生理学 | 前负荷/后负荷、心肌重构 | 慢性心衰、急性心衰 |
| 抗心律失常药 | 药理学 | 钠/钾/钙通道阻滞、β受体阻滞 | 心律失常治疗 |
| 抗高血压药 | 药理学 | ACEI/ARB、CCB、利尿剂 | 高血压治疗 |
| 抗心衰药 | 药理学 | ACEI/ARB、β阻滞剂、利尿剂 | 心衰治疗 |

**PBL案例设计要点：**
- 以胸痛/心悸/呼吸困难为切入点
- 桥接解剖（冠脉/传导系统）→ 生理（电生理/泵血）→ 病理（粥样硬化/缺血）→ 药理（治疗药物）
- 检查结果：心电图、心肌标志物、超声心动图、冠脉造影

## 三、跨学科整合案例的知识图谱设计

### 3.1 知识图谱设计流程

1. **确定核心临床问题：** 选择一个临床场景作为中心
2. **识别涉及学科：** 列出所有相关基础医学学科
3. **标注知识节点：** 每个学科中与案例相关的核心概念
4. **绘制关联线：** 标注学科间知识点的关联关系
5. **标注权重：** 根据教学目标标注各知识点的重要性
6. **识别盲区：** 标注学生可能的知识盲区和难点
7. **设计锚点：** 将知识图谱中的关键节点设计为案例的知识锚点

### 3.2 知识图谱可视化示例

以"糖尿病酮症酸中毒"为例：

```
[临床问题] 糖尿病酮症酸中毒
         |
    +----+----+----+----+
    |         |    |    |
[生化]    [病理] [药理] [病理生理]
 糖代谢   β细胞  胰岛素  渗透压
 脂代谢   损伤   治疗    酸碱失衡
 酮体    血管   降糖药  电解质
    |         |    |    |
    +----+----+----+----+
         |
    [免疫学]
    自身免疫
    1型DM机制
```

### 3.3 学科权重分配

| 难度层级 | 主导学科 | 支撑学科 | 拓展学科 |
|---------|---------|---------|---------|
| L1 | 1-2个（占50%） | 1-2个（占30%） | 0-1个（占20%） |
