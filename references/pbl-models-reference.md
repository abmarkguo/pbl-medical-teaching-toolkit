# PBL 模型详细对比与操作指南

## 概述

本文档对三种国际主流 PBL 模型进行详细对比，提供操作层面的步骤指南、适用场景分析和融合策略，供教师根据学科特点和学生水平选择最佳模型。

---

## 一、马斯特里赫特七步法（Maastricht 7-Step Model）

### 1.1 起源与特点

- **起源：** 荷兰马斯特里赫特大学医学院（1974年）
- **理论基础：** 建构主义学习理论、认知心理学
- **核心特征：** 以问题为起点，通过七步结构化流程引导学生从已知到未知的自主探究
- **国际地位：** 全球应用最广泛的 PBL 流程模型，被 WHO 推荐为医学教育改革方向

### 1.2 七步流程详解

#### Step 1：澄清术语和概念（Clarify Terms and Concepts）
- **目标：** 确保全体学生对案例中的术语和概念有共同理解
- **操作：**
  - 导师朗读或分发案例第一幕
  - 学生标记不熟悉的术语
  - 小组讨论澄清术语含义
  - 导师确认理解，纠正误解
- **时长：** 5-10分钟
- **导师要点：** 不要在此步过度展开，仅澄清概念即可

#### Step 2：定义问题（Define the Problem）
- **目标：** 明确案例需要解决的核心问题
- **操作：**
  - 小组讨论：这个案例的核心问题是什么？
  - 用一句话表述问题（Problem Statement）
  - 可以定义多个子问题
- **时长：** 5-10分钟
- **导师要点：** 引导学生从患者角度和医学角度两个层面定义问题

#### Step 3：头脑风暴/激活已有知识（Brainstorm / Activate Prior Knowledge）
- **目标：** 激活学生的已有知识，生成对问题的初步解释
- **操作：**
  - 自由讨论：基于已有知识，对问题有什么想法？
  - 记录所有观点，不做评判
  - 鼓励联想和猜测
  - 使用白板或共享文档记录
- **时长：** 15-20分钟
- **导师要点：** 不纠正错误观点，鼓励发散思维，记录所有想法

#### Step 4：结构化整理（Structure and Organize Ideas）
- **目标：** 将头脑风暴产生的观点进行分类和组织
- **操作：**
  - 将观点按主题/系统/机制分类
  - 寻找观点之间的关联
  - 形成初步的解释框架
  - 识别知识盲区（"我们不知道什么"）
- **时长：** 10-15分钟
- **导师要点：** 引导学生发现知识缺口，为下一步制定学习目标做铺垫

#### Step 5：制定学习目标（Formulate Learning Objectives）
- **目标：** 将知识盲区转化为可操作的学习目标
- **操作：**
  - 基于Step 4识别的知识盲区，制定学习目标
  - 学习目标应具体、可测量、可达成
  - 使用Bloom分类法标注认知层次
  - 每组通常生成5-8个学习目标
  - 导师审核学习目标，确保覆盖核心知识
- **时长：** 10-15分钟
- **导师要点：** 确保学习目标不过宽或过窄，覆盖案例设计的核心知识锚点

#### Step 6：自主学习（Self-Study）
- **目标：** 学生围绕学习目标进行自主学习
- **操作：**
  - 学生独立查阅教材、文献、数据库
  - 整理学习笔记和发现
  - 准备小组汇报材料
  - 可小组分工，每人负责部分学习目标
- **时长：** 2-5天（课外）
- **导师要点：** 不干预过程，但可提供资源建议

#### Step 7：汇报与综合（Report and Synthesize）
- **目标：** 学生汇报自主学习成果，小组综合形成共识
- **操作：**
  - 各成员汇报学习成果
  - 小组讨论，整合信息
  - 回到案例，用新知识重新解释问题
  - 反思学习过程和团队协作
  - 导师总结和补充
- **时长：** 60-90分钟
- **导师要点：** 引导知识整合而非简单汇报，促进深层次理解

### 1.3 适用场景

| 条件 | 适用度 |
|------|--------|
| 本科低年级 | ★★★★★ |
| 本科高年级 | ★★★★☆ |
| 研究生/规培 | ★★★☆☆ |
| 首次接触PBL的学生 | ★★★★★ |
| 需要建立自主学习习惯 | ★★★★★ |
| 课时充裕（3次以上） | ★★★★★ |

## 二、夏威夷 PBL 模型（Hawaiian PBL Model）

### 2.1 起源与特点

- **起源：** 美国夏威夷大学 John A. Burns 医学院（1990年代）
- **理论基础：** 认知负荷理论、情境学习理论
- **核心特征：** 强调早期临床推理训练，通过"触发器"（Trigger）材料激发学习，四阶段循环递进
- **与七步法的区别：** 更早引入临床推理，减少结构化步骤，增加临床情境感

### 2.2 四阶段流程详解

#### Phase 1：遭遇与识别（Encounter & Identify）
- **目标：** 学生接触临床情境，识别需要解决的核心问题
- **操作：**
  - 呈现案例触发器（患者故事/视频/模拟场景）
  - 学生沉浸式阅读/观察
  - 识别关键信息和异常表现
  - 提出初步问题："这里发生了什么？"
  - 记录第一印象和直觉判断
- **时长：** 20-30分钟
- **导师要点：** 营造临床情境感，鼓励学生像医生一样思考

#### Phase 2：解释与推理（Explain & Reason）
- **目标：** 运用基础医学知识解释临床表现，生成鉴别诊断假设
- **操作：**
  - 基于已有知识解释症状和体征的机制
  - 生成鉴别诊断假设列表
  - 讨论各假设的支持和反对证据
  - 识别需要进一步学习的知识领域
  - 形成"待探究问题清单"
- **时长：** 30-40分钟
- **导师要点：** 引导机制层面的解释（"为什么会出现这个症状？"），促进基础-临床桥接

#### Phase 3：应用与整合（Apply & Integrate）
- **目标：** 应用自主学习获得的新知识，整合多学科信息
- **操作：**
  - 自主学习期后（2-3天），学生汇报新知识
  - 将新知识应用于鉴别诊断假设的验证
  - 整合基础医学和临床医学知识
  - 更新假设列表，排除或确认假设
  - 讨论检查结果对诊断的提示
- **时长：** 40-50分钟
- **导师要点：** 引导知识整合而非简单汇报，关注跨学科链接

#### Phase 4：评估与反思（Evaluate & Reflect）
- **目标：** 验证最终诊断，反思学习过程
- **操作：**
  - 案例揭示最终诊断
  - 对比学生假设与实际诊断
  - 讨论诊疗方案和循证依据
  - 反思推理过程中的偏差和盲区
  - 总结学习收获和待改进之处
- **时长：** 20-30分钟
- **导师要点：** 引导元认知反思，关注"怎么想的"而非"想对了没有"

### 2.3 适用场景

| 条件 | 适用度 |
|------|--------|
| 本科低年级 | ★★☆☆☆ |
| 本科高年级 | ★★★★★ |
| 研究生/规培 | ★★★★☆ |
| 需要强化临床推理 | ★★★★★ |
| 基础-临床桥接课程 | ★★★★★ |
| 课时中等（2-3次） | ★★★★☆ |

## 三、中国本土化 PBL 模型

### 3.1 起源与特点

- **起源：** 综合 上海交通大学医学院、北京大学医学部、四川大学华西医学中心、复旦大学上海医学院 等国内顶尖医学院校的 PBL 教学实践经验
- **理论基础：** 融合建构主义与中国医学教育传统（重视知识系统性）
- **核心特征：** 适应中国医学院校的大班教学实际（8-12人/组），两课时三阶段结构，兼顾系统性和探究性
- **与国外模型的区别：** 更适应中国教学节奏，强化教师引导作用，保留知识系统性

### 3.2 三阶段流程详解

#### 第一课时：案例呈现与头脑风暴

**阶段A：案例呈现与初步讨论（约40分钟）**
- 教师分发案例第一幕
- 学生独立阅读5分钟
- 小组讨论：案例中有什么问题？可能的原因？
- 头脑风暴：记录所有想法和假设
- 教师引导：确保讨论覆盖多个角度

**阶段B：知识梳理与目标制定（约40分钟）**
- 将头脑风暴结果分类整理
- 识别知识盲区
- 制定学习目标（5-8个）
- 教师审核学习目标的合理性
- 分配自主学习任务

**阶段C：自主学习安排（约10分钟）**
- 明确学习目标和研究方向
- 分配小组成员的研究任务
- 约定汇报方式和时间
- 推荐学习资源和文献

#### 自主探究间隔期（2-5天）

- 学生围绕学习目标进行独立学习
- 查阅教材、文献、数据库
- 整理学习笔记
- 准备汇报材料（PPT/文档/思维导图）
- 小组可在线交流讨论
- 教师不干预，但可答疑

#### 第二课时：小组汇报与综合

**阶段D：小组汇报（约40分钟）**
- 各成员汇报自主学习成果
- 每人5-8分钟（含2分钟提问）
- 释放案例第二幕和第三幕信息
- 对照新信息检验学习成果

**阶段E：综合讨论与整合（约30分钟）**
- 整合各成员的学习成果
- 用新知识重新解释案例问题
- 讨论诊断和治疗方案
- 教师引导跨学科知识整合

**阶段F：反思与总结（约20分钟）**
- 学生反思学习过程和收获
- 教师总结核心知识点
- 评价小组合作效果
- 布置后续学习任务

### 3.3 适用场景

| 条件 | 适用度 |
|------|--------|
| 本科低年级 | ★★★★☆ |
| 本科高年级 | ★★★★★ |
| 研究生/规培 | ★★★☆☆ |
| 大班额（8-12人/组） | ★★★★★ |
| 课时受限（2次课时） | ★★★★★ |
| 中国医学院校常规教学 | ★★★★★ |

## 四、三模型融合策略

### 4.1 何时融合

- 学生已有一定PBL经验，需要提升临床推理能力时
- 基础-临床桥接课程，需要兼顾知识建构和临床思维时
- 不同教学阶段使用不同模型（如低年级用七步法，高年级用夏威夷模型）

### 4.2 融合方案示例

**方案A：七步法 + 夏威夷模型**
- Step 1-4 采用七步法的结构化流程
- Step 3-5 融入夏威夷模型的临床推理元素（生成假设、验证假设）
- Step 6-7 采用夏威夷模型的整合与反思阶段
- 适用于本科高年级的基础-临床桥接课程

**方案B：中国本土化 + 七步法**
- 课时1采用中国本土化模型的案例呈现和头脑风暴
- 在知识梳理阶段融入七步法的Step 4-5（结构化整理和学习目标制定）
- 课时2采用中国本土化模型的小组汇报与综合
- 适用于首次接触PBL的本科低年级

**方案C：三模型完全融合**
- 第一课时：中国本土化的案例呈现 + 七步法的Step 1-5
- 自主学习期：夏威夷模型的自主学习导向（以临床推理问题为驱动）
- 第二课时：夏威夷模型的Phase 3-4 + 中国本土化的综合讨论
- 适用于有经验的PBL教师和有一定经验的学生

### 4.3 融合注意事项

1. **保持流程清晰：** 融合后仍需让学生明确当前所处的阶段和任务
2. **避免过度复杂：** 融合不是简单叠加，而是取各模型所长
3. **循序渐进：** 建议从单一模型开始，逐步引入其他模型元素
4. **评估适配：** 评估方式需与所选模型匹配

## 五、模型选择的决策流程

```
1. 学生是否首次接触PBL？
   → 是 → 推荐马斯特里赫特七步法
   → 否 → 继续

2. 教学目标是否侧重临床推理？
   → 是 → 推荐夏威夷模型
   → 否 → 继续

3. 课时是否受限（仅2次课时）？
   → 是 → 推荐中国本土化模型
   → 否 → 继续

4. 是否为基础-临床桥接课程？
   → 是 → 推荐七步法+夏威夷融合方案
   → 否 → 推荐中国本土化模型

5. 学生是否为研究生/规培生？
   → 是 → 推荐夏威夷模型或融合方案C
   → 否 → 根据上述条件选择
```
