# PBL 难度矩阵模板

> 本模板提供PBL案例的难度分级标准矩阵，帮助教师根据学段和教学目标选择或设计合适难度的案例。

---

## 一、难度分级总览

| 维度 | L1 本科低年级 | L2 本科高年级 | L3 研究生/规培 |
|------|-------------|-------------|--------------|
| 适用学段 | 医学本科1-3年级 | 医学本科4-5年级 | 研究生/住院医师规培 |
| 主导学科 | 解剖、生理、生化 | 病理、病理生理、药理 | 跨学科整合、循证医学 |
| 案例复杂度 | 单一系统、典型表现 | 多系统关联、需鉴别 | 复杂合并症、不典型表现 |
| 信息量（每幕） | 2-3页 | 3-4页 | 4-5页 |
| 检查结果复杂度 | 常规检查为主 | 含影像学和特殊检查 | 含矛盾结果和鉴别难点 |
| 鉴别诊断数量 | 2-3个 | 3-5个 | 5个以上 |
| 知识锚点数量 | 9-12个 | 12-18个 | 15-24个 |
| 跨学科比例 | 20-30% | 30-50% | 40-60% |
| 自主学时 | 4-6学时 | 6-8学时 | 8-12学时 |
| Bloom层次 | L1-L3 | L3-L5 | L4-L6 |

---

## 二、各维度详细标准

### 2.1 案例复杂度矩阵

| 复杂度要素 | L1 | L2 | L3 |
|-----------|----|----|-----|
| 涉及系统数 | 1个 | 2-3个 | 3个以上 |
| 疾病表现典型性 | 典型表现 | 基本典型，含变异 | 不典型，含伪装 |
| 合并症数量 | 0-1个 | 1-2个 | 2-3个以上 |
| 并发症数量 | 0-1个 | 1-2个 | 2个以上 |
| 干扰信息量 | 少量（1-2条） | 适量（2-4条） | 较多（4-6条） |
| 诊疗争议性 | 无 | 轻微 | 明显 |
| 伦理困境 | 简单提及 | 嵌入讨论 | 核心讨论点 |
| 时间跨度 | 单次就诊 | 数天至数周 | 数月至数年 |

### 2.2 检查结果复杂度矩阵

| 检查类型 | L1 | L2 | L3 |
|---------|----|----|-----|
| 血常规/尿常规 | 简单异常 | 多项异常 | 异常+矛盾 |
| 生化检查 | 1-2项异常 | 3-5项异常 | 多项异常+动态变化 |
| 影像学 | X线/超声描述 | CT/MRI描述 | 多种影像对比+不典型表现 |
| 心电图/脑电图 | 基本异常 | 复杂异常 | 需鉴别分析的异常 |
| 病理检查 | 不涉及或简单描述 | 基本病理描述 | 详细病理+免疫组化/分子 |
| 微生物学 | 不涉及或简单培养 | 培养+药敏 | 复杂病原体+耐药分析 |
| 特殊检查 | 不涉及 | 1-2项 | 多项+结果矛盾 |
| 检查结果一致性 | 完全一致 | 基本一致 | 部分矛盾需分析 |

### 2.3 临床推理要求矩阵

| 推理要素 | L1 | L2 | L3 |
|---------|----|----|-----|
| 症状机制解释 | 单一机制 | 多因素机制 | 复杂交互机制 |
| 鉴别诊断深度 | 2-3个方向 | 3-5个含排除依据 | 5个以上含循证评价 |
| 假设验证 | 简单验证 | 多步验证 | 迭代验证+修正 |
| 检查结果解读 | 直接对应 | 需机制解释 | 需综合分析+排除假阳性/假阴性 |
| 诊断推理链 | 2-3步 | 4-5步 | 6步以上+分支 |
| 治疗决策 | 标准方案 | 个体化调整 | 循证评价+个体化+替代方案 |
| 预后判断 | 简单预测 | 多因素分析 | 不确定性分析+长期管理 |

## 四、难度调整指南

### 4.1 升级难度（L1→L2→L3）

**L1→L2 调整：**
- 增加合并症或并发症
- 增加检查结果的复杂度
- 增加鉴别诊断数量
- 提高Bloom层次（更多分析/评价）
- 增加跨学科知识锚点
- 增加干扰信息

**L2→L3 调整：**
- 使用不典型临床表现
- 增加诊疗争议和伦理困境
- 增加检查结果的矛盾性
- 提高推理链复杂度
- 增加循证医学评价要求
- 增加长期管理和个体化决策

### 4.2 降级难度（L3→L2→L1）

**L3→L2 调整：**
- 简化临床表现为典型表现
- 减少合并症和并发症
- 减少干扰信息
- 简化检查结果
- 降低Bloom层次
- 减少跨学科比例

**L2→L1 调整：**
- 使用单一系统疾病
- 仅保留常规检查
- 减少鉴别诊断至2-3个
- 聚焦基础概念理解
- 减少自主学时
- 使用七步法模型

## 五、难度验证检查清单

### 5.1 难度匹配验证

- [ ] 案例复杂度与目标学段匹配
- [ ] 检查结果复杂度与学生知识水平匹配
- [ ] 学习目标的Bloom层次分布合理
- [ ] 知识锚点的学科分布合理
- [ ] 自主学时与案例难度匹配
- [ ] 鉴别诊断数量与推理要求匹配

### 5.2 难度梯度验证

- [ ] 第一幕信息量适中（不过多也不过少）
- [ ] 第二幕信息量较第一幕增加
- [ ] 第三幕聚焦综合而非新信息
- [ ] 学习目标从低阶到高阶递进
- [ ] 引导问题认知层次递进
- [ ] 知识锚点从单学科到跨学科递进
